# 싱크탱크 의사결정 지원 플랫폼 — 목업 설계 (개정 2)

날짜: 2026-07-23
단계: 클릭 가능한 목업 (목 데이터 / 목 분석, 백엔드 없음)

## 1. 제품 비전 (전체 — 지금 다 만들지 않음)

거시 이슈에 대해 유저가 분기 시나리오를 놓고 토론·지지(star)하고, 시간이 흐르며 현실과 부합한 시나리오가 현실화되어 **결정 흐름**을 이룬다. 현실화·반영된 시나리오는 디지털 트윈 지도와 AI 다관점 분석으로 의사결정을 돕는다.

4개 독립 서브시스템:
1. **이슈 큐레이션 엔진** — 뉴스·SNS·싱크탱크 보고서·공공발표 취합 → AI가 거시 이슈 선정
2. **시나리오 토론·지지 플랫폼** — 유저가 분기 시나리오 제출 → GitHub 이슈식 토론 + star 지지 → 토론 요약·근거 저장
3. **디지털 트윈 지도** — 반영된 시나리오를 지도에서 확인, 현실 상태 동기화(목)
4. **AI 영향·결과 분석** — 시나리오 + 근거 → 정치·경제·군사·민간 4관점 예측

## 2. 이번 단계 범위

**클릭 가능한 목업.** 전체 흐름을 관통하는 정적 프로토타입. 실제 수집·LLM·인증·영속 저장·현실 데이터 피드 **없음**. 모든 데이터·분석·시간 진행·현실 동기화는 **목**.

목표: 흐름·화면·개념(특히 두 층 분리와 결정 흐름)을 눈으로 검증.

## 3. 핵심 개념 — 세 상태 완전 독립

혼동 방지를 위해 셋을 독립 상태로 못박는다:

| 상태 | 의미 | 쓰임 | 판정 |
|------|------|------|------|
| **star** (int) | 유저 지지 지표 | 트리 가지치기(상위만 표시)·정렬 | 유저가 토론 화면에서 star 토글 |
| **realizedAt** (날짜\|null) | 현실과 부합해 현실화된 시점 | 타임라인 트리의 **강조 줄기**(결정 흐름) | 목 데이터에 하드코딩된 시점 |
| **reflected** (bool) | 디지털 트윈 지도 표시 여부 | 지도 핫스팟 | 토론 화면에서 토글 |

- **지도 핫스패 기준 = `reflected`** (realizedAt·star와 무관).
- 세 상태는 서로 종속되지 않음. 아무 시나리오나 star 받을 수 있고, realizedAt 없어도 reflected 가능.

## 4. 스택

- 순수 HTML / CSS / 바닐라 JS. 빌드 없음, 프레임워크 없음.
- 목 데이터는 `web/data.js` 하나에 전역 `window.DB`로. `MOCK_NOW = '2026-07-23'` 고정, 모든 `realizedAt`은 그 이전.
- 화면당 정적 HTML 파일 증식 금지 — 레코드 화면은 단일 템플릿 + `?id=` 쿼리 파라미터.
- **타임라인 트리 그래프는 라이브러리 없이 SVG 직접 그림.** treeX는 날짜(`createdAt`/`realizedAt`)에서 계산, treeLane(y 레인 정수)만 목 데이터에 하드코딩. 트리 레이아웃 알고리즘·줌/팬/드래그 없음.
- 상태 지속은 localStorage 오버라이드(star / reflected / 편집 / lastVisit). 실제 저장소 없음.

## 5. 화면 (6 + 공용 셸)

**공용 셸:** 상단 고정 nav `[대시보드] [지도]` + 로그인 유저 배지(목).

1. **이슈 대시보드** (홈, `index.html`) — AI 선정 거시 이슈 카드. 제목·요약·출처·트래픽·heat 배지 + 시나리오 수. 클릭 → 이슈 타임라인.
2. **이슈 타임라인 트리 그래프** (`issue.html?id=`) — 이번 개정의 중심.
   - SVG 트리: **x = 시간**(createdAt 날짜→px 매핑), **y = treeLane**. 부모-자식 엣지.
   - **현실화 줄기 강조** (realizedAt ≤ 커서): 진한 색/굵은 엣지 = 결정 흐름. 미현실 노드는 흐리게.
   - **슬라이더**: 커서 시점을 현재↔과거 이동. 커서보다 늦게 생성된 노드 숨김, 커서 이후 현실화는 미현실로 표시.
   - **애니메이션**: `lastVisit`이 있고 `MOCK_NOW`보다 과거면, 페이지 열 때 커서를 `lastVisit`→`MOCK_NOW`로 이산 스윕(setInterval + CSS transition). 없으면 생략, 커서 최신 시작.
   - **가지치기**: 커서 기준 미현실 분기는 현실화 줄기로부터 **depth ≤ 2**, 형제 중 **star 상위 2개만** 표시.
   - 노드 클릭 → 시나리오 토론. "+ 시나리오 제출" 버튼.
   - **빈 트리**(시나리오 0개, 예 i4): "아직 분기 없음" 안내 + 슬라이더 숨김. **realizedAt 전부 null**(예 i3): 강조 줄기 없이 전부 후보 표시 + "아직 현실화된 분기 없음" 안내.
3. **시나리오 토론** (`scenario.html?id=`) — 본문 · "X에서 분기됨" breadcrumb · **토론 요약**(discussionSummary) · **근거**(rationale) 박스 · **star 버튼**(토글, 카운트) · **모더레이션 필터된 코멘트**(금지어 마스킹) · **지도 반영 토글**(reflected) · **인라인 편집**(제목·본문 수정 → localStorage → 지도·분석 반영) · 하위 분기 목록.
4. **시나리오 제출 폼** (`submit.html?issue=`) — 제목·설명·근거 입력 + 부모 시나리오 선택. 제출은 목 — alert 후 이슈 타임라인 복귀(트리에 실제 추가되진 않음).
5. **디지털 트윈 지도** (`map.html`) — `reflected` 시나리오 핫스팟 + **현실 동기화 상태 배지**(목: "마지막 동기화 …"). 핫스팟 클릭 → 패널에 **토론 요약 + 근거** 노출 + "AI 분석 실행". 반영 0개 빈 상태.
6. **AI 분석 뷰** (`analysis.html?id=`) — 정치·경제·군사·민간 4관점 패널(목 분석, **편집 반영된 본문 기준**) + "분석 중…" 연출.

## 6. 데이터 모델 (목, `web/data.js`)

- `DB.now`: `'2026-07-23'` (MOCK_NOW)
- `DB.badwords`: `['젠장','바보','멍청']` (목 모더레이션 금지어)
- **issue**: `id, title, summary, sources[], traffic, heat`
- **scenario**: `id, issueId, parentScenarioId|null, title, body, rationale, discussionSummary, stars, createdAt, realizedAt|null, treeLane(int), reflected(bool), mapLocation{x,y,label}|null`
- **comment**: `id, scenarioId, author, body, ts` (금지어 포함 샘플 1개 이상)
- **analysis**: `scenarioId, lenses{political, economic, military, civilian}`

제약: `realizedAt`은 있으면 `createdAt` 이후·`DB.now` 이전. `reflected`면 `mapLocation` 필수. 최소 한 이슈(i1)는 depth ≥ 3, star 분산이 커야 가지치기 시연 가능.

## 7. 흐름

대시보드 → 이슈 타임라인(트리, 슬라이더/애니) → 노드 클릭 → 시나리오 토론(star·편집·반영 토글) → 지도(반영 확인, 요약·근거) → AI 분석 4관점. 대시보드/지도는 nav로 항상 접근.

## 8. Out-of-scope (이번 단계 제외)

- 실제 인증/로그인, 대댓글, 실제 ML 모더레이션(금지어 배열 마스킹만)
- 실제 GIS 지도, 실시간 현실 데이터 피드/동기화(상태 배지·트리 애니는 전부 목)
- 트리 그래프 레이아웃 알고리즘·줌/팬/노드 드래그·라이브러리 (하드코딩 lane + 날짜 매핑만)
- 실제 뉴스 수집·LLM·영속 저장 (localStorage 오버라이드만)

## 9. 검증 (목업이라도)

- 6개 화면 링크 도달·흐름 무단절.
- 두 층 분리 보임: star 카운트 변화, realizedAt 강조 줄기, reflected 지도 반영이 서로 독립 동작.
- 트리: 슬라이더로 과거 이동 시 노드 숨김/미현실화, 가지치기(depth·star) 동작, lastVisit 애니 재생/생략.
- 편집한 본문이 지도 패널·AI 분석에 반영.
- 빈 상태: 빈 트리(i4), 현실화 0(i3), 지도 반영 0.
- 모더레이션: 금지어 코멘트 마스킹.
- `test/check-data.mjs` 무결성 통과.
